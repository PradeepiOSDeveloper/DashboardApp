📱 Dashboard iOS App (MVVM + UIKit)

🚀 Overview

This project is a Dashboard screen implementation based on provided Figma design.

⸻

🧱 Architecture

The project follows MVVM (Model-View-ViewModel):
	•	Model → Data layer
	•	ViewModel → Business logic
	•	View (UIKit) → UI rendering

⸻

⚙️ Features
	•	Clean architecture
	•	Modular structure
	•	Mock API integration
	•	Reusable components
	•	Smooth UI performance

⸻

🛠 Tech Stack
	•	Swift
	•	UIKit
	•	MVVM
	•	AutoLayout

⸻

📡 API
	•	Mock JSON used
	•	Easily replaceable with real API

⸻

▶️ How to Run
	1.	Clone repo
	2.	Open .xcodeproj
	3.	Run on simulator

⸻

📁 Folder Structure
	•	Core → Network & Utils
	•	Modules → Feature modules (Dashboard)
	•	Resources → Assets

⸻

🎯 Highlights
	•	No UI blocking
	•	Scalable architecture
	•	Clean separation of concerns
